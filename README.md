# 🎓 College Admission Form

A simple College Admission Form created using only HTML.

## 📌 Features

- Student Name
- Username & Password
- Phone Number
- Date of Birth
- Gender Selection
- Address
- Year Selection
- Branch Selection
- Graduation Year
- Skills
- Hobbies
- Feedback Section
- Submit & Reset Buttons

## 🛠️ Technologies Used

- HTML5

## 🎯 Purpose

This project was created to practice HTML forms and improve my understanding of basic web development concepts.

- Add CSS for better styling
- Make the form responsive
- Add JavaScript form validation
